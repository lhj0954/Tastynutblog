# Tastynut's blog
혼자서 만들고 운영해보는 개인 웹사이트입니다.

접속하기: [Tastynut-blog](http://www.tastynut.blog) (aws프리티어가 끝나서 종료) 

<hr/>

## 기술 스택
백엔드: SpringBoot, JPA, Oauth2.0(google, kakao, naver), SpringSecurity, Jwt

DB: MariaDB

프론트엔드: React.js

배포: AWS, Docker, DNS
